A detailed changelog of minecraft versions with a particular focus on discontinued features.

- Changes marked with ★ indicate the start version of a discontinued feature.
- Changes marked with ☆ indicate that they made a feature discontinued (First version where the feature is unobtainable).
- Changed marked with ⬟ and ⬠ are analogous to ★ and ☆, but with steps of complex version path discontinued features instead of regular DFs.
